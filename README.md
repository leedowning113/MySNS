# MySNS

Java와 JSP/Servlet을 이용해 구현한 SNS 웹 프로젝트입니다.  
회원가입, 로그인, 피드 작성, 친구 관리 등의 기본 기능을 구현했으며, 이후 WAR 파일로 배포하여 AWS EC2 환경의 Tomcat 서버에서도 실행해보았습니다.

## 사용 기술
- Java
- JSP / Servlet
- MySQL
- HTML / CSS / JavaScript
- Apache Tomcat
- AWS EC2

## 주요 기능
- 회원가입 / 로그인 / 로그아웃
- 회원정보 조회 및 수정
- 회원탈퇴
- 피드 작성 및 목록 조회
- 친구 추가 / 조회 / 삭제
- 세션 기반 로그인 상태 유지

## 프로젝트 설명
이 프로젝트는 Java 웹 개발의 기본 구조를 이해하고, JSP와 Servlet, DAO를 이용한 데이터 처리 흐름을 익히기 위해 진행했습니다.  
또한 로컬 환경에서 개발한 프로젝트를 WAR 파일로 배포하고, AWS EC2의 Tomcat 서버에서 실행해보며 배포 과정도 함께 경험했습니다.

## 느낀 점
기능 구현뿐 아니라 DB 연동, 세션 처리, 서버 배포까지 직접 해보면서 웹 서비스가 동작하는 전체 흐름을 이해할 수 있었습니다.
